Project 1: Maven-Based BDD Automation Framework

This project is a comprehensive Behavior-Driven Development (BDD) automation framework built using Maven, Java, Selenium WebDriver, and Cucumber. It targets the OpenCart demo website (https://demo.opencart.com/) and follows best practices like the Page Object Model (POM) and object-oriented programming (OOP) principles.

Key Features

  • Behavior-Driven Development (BDD): Test scenarios are written in plain English using Gherkin syntax, making them understandable to both technical and non-technical stakeholders.
  • Page Object Model (POM): The framework uses the POM design pattern to encapsulate UI elements and actions, improving code maintainability and reusability.
  • Cross-Browser Testing: Supports running tests on multiple browsers (Chrome and Firefox) by configuring the `config.properties` file.
  • Continuous Integration/Continuous Delivery (CI/CD) Ready: Includes a `Jenkinsfile` for seamless integration with Jenkins pipelines, enabling automated test execution as part of the CI/CD process.
  • Detailed Reporting: Generates HTML and JSON reports to provide clear and detailed test results.
  • Reusable Utilities: Includes utility classes for configuration management, WebDriver initialization, and various wait strategies.
  • Object-Oriented Programming (OOP): Implements OOP principles like encapsulation, inheritance, polymorphism, and abstraction to enhance the framework's robustness and scalability.

Technologies Used

  • Java: Core programming language.
  • Maven: Build and dependency management.
  • Selenium WebDriver: Browser automation.
  • Cucumber: BDD framework for writing tests.
  • JUnit: Test execution.
  • Jenkins: CI/CD integration.

Project Structure

  • src/main/java/com/example/: Contains the Page classes (e.g., HomePage, LoginPage) and utility classes.
  • src/test/java/com/example/: Contains the feature files, step definitions, and runners.

Setup Instructions

  1. Install Java 11 or higher.
  2. Install Maven.
  3. Install ChromeDriver and/or GeckoDriver (Firefox).
  4. Install Jenkins (optional, for CI/CD).
  5. Clone the repository: git clone https://github.com/your-repo/bdd-automation-framework.git
  6. Navigate to the project directory: cd bdd-automation-framework
  7. Install dependencies: mvn clean install
  8. Update Configuration: Edit the config.properties file in src/main/resources.

Running Tests

  • Run all tests: mvn test
  • Run tests with specific tags: mvn test -Dcucumber.filter.tags="@SmokeTest"

View Reports

  • After running the tests, view the HTML report at: target/cucumber-reports.html
